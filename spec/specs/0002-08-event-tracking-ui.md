# 0002-08 Event Tracking UI Specification

## Status
Open

## Overview
This document outlines the UI requirements for recording agricultural events, based on Feature 8 of PRD 0002.

## Requirements

1. **Event Tracking Forms**
   - Provide comprehensive forms to record agricultural events for individual fields.
   - Event types to include:
     - Planting
     - Fertiliser application
     - Slurry application
     - Spraying
     - Harvesting
     - Tilling

## Technical Details
- Must be built using Angular and Angular Material.
- Forms must be accessible and responsive for field use on mobile devices.
