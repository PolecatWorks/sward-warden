# 0007-01 Digital Pesticide Records Export Specification

## Status
Complete

## Overview
This document specifies the requirements for generating digital exports of pesticide records, based on Feature 1 of PRD 0007.

## Requirements

1. **Machine-Readable Format**
   - Provide the ability to export Plant Protection Product (PPP) application records.
   - The export must be in a "machine-readable" digital format suitable for electronic submission to DAERA.

2. **Required Data Points**
   - The export must include the following information for each record:
     - MAPP (Product authorization number)
     - EPPO crop/land-use code
     - BBCH growth-stage code
