# 0004-03 Fertiliser Application Records Specification

**State**: Complete

## Scope
This specification covers the fertiliser application records required to maintain compliance with NAP regulations (PRD 0004).

## Features
- Ability to record the type and amount of nitrogen fertiliser applied.
- Include calculations or inputs for nitrogen content of organic manures.
- Store evidence of grazing rights on common land or control of agricultural areas.

## Data Model Requirements
- Entity for `FertiliserApplications`.
- Extend `Events` or link.
