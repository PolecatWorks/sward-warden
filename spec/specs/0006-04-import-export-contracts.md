# 0006-04 Import and Export Contracts Specification

## Status
Complete

## Overview
This document outlines the requirements for tracking import and export contracts for manure, based on Feature 4 of PRD 0006.

## Requirements

1. **Contractual Documentation**
   - Provide a system to document written contractual agreements for the movement of manure onto or off the farm.

2. **Contract Details**
   - Track the following data points for compliance reporting:
     - Origin
     - Destination
     - Volume
     - Contract length
